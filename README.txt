Este repositorio contiene el código para generar la web estática
http://procedimientoslibres.es.

Para la generación se utiliza Pyll: https://github.com/arthurk/pyll

Instalación
-----------

- Instale Pyll: https://github.com/arthurk/pyll
- Instale las fuentes de web-procedimientoslibres.es

Sitúese en el directorio de las fuentes y ejecute el comando pyll. Se
generará un directorio _output donde se hallará el código de la web
estática.

Asimismo, ejecute el comando pyll --server en el directorio donde se
hallan las fuentes y podrá acceder a la web en http://localhost:8000
